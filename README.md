# AI Fitness App

An AI-based fitness application that uses computer vision to track workouts,
provide form correction, and visualize user progress.

## Tech Stack
- Python
- Django
- Supabase (PostgreSQL)
- Computer Vision (Pose Detection)
- HTML, CSS, JavaScript

## Features
- Real-time exercise tracking
- Form correction & feedback
- Progress visualization
- Meal planning
- AI coach module
